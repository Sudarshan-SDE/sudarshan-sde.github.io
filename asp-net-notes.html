Introduction to ASP.NET

ASP.NET is a web development framework by Microsoft used to build dynamic web applications, APIs, and enterprise systems using C#.

ASP.NET MVC

MVC stands for Model-View-Controller. It separates application logic into three components for better structure and maintainability.

ASP.NET Core

ASP.NET Core is a cross-platform, high-performance framework for building modern web apps and APIs.

Routing

Routing maps URLs to controllers and actions in ASP.NET applications.

Middleware

Middleware is a pipeline component that handles requests and responses in ASP.NET Core.

Web API

ASP.NET Core Web API lets you build RESTful HTTP services with automatic model validation.

Entity Framework Core

EF Core is an ORM for .NET that lets you interact with databases using C# objects and LINQ.

Authentication

ASP.NET Core supports cookie auth, JWT bearer tokens, OAuth2, and OpenID Connect.
